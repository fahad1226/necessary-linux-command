** to increase swap memory

-> swapon --show  (to show the current swap file size)
-> ls -lh /swapfile ( show the swap file soze in human readbel form)
-> sudo swapoff /swapfile (to remove the current swap memory)
-> sudo fallocate -l 4G /swapfile (now increase the size)
-> sudo mkswap /swapfile (set the new swap memroy)
-> swapon --show (see the file increrases to 4GB)
